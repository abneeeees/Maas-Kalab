# Maas Kalab

A very serious website for very unserious meat activities.

`Maas Kalab` is a community for non-vegetarian **Maas (Non-Veg)** Lovers who love to eat and cook **Maas**.

## What we Love

- Maas
- Maas Hacks
- Maas Lovers

## Run It

```bash
bun run dev
npm run dev
pnpm run  dev
```

Then open `http://localhost:3000`
